# Java-Battle-Game
A Java text-based battle game that uses object-oriented programming to simulate player vs. creature combat with abilities, health management, and power-up mechanics.

The game includes character management, ability selection, random enemy attacks, and power-ups
that modify player stats during combat.

## Features
- Player vs creature combat system
- Ability-based attacks with different damage values
- Character selection and removal
- Health regeneration mechanics
- Randomized enemy abilities
- Power-up system

## Concepts Used
- Java Object-Oriented Programming
- Inheritance
- ArrayLists
- User input with Scanner
- Game logic and control flow
